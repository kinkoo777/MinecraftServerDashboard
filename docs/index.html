Download & unzip

Grab the ZIP, then right-click it and choose Extract All. Put the folder anywhere you like.

2

Double-click to launch

On Windows, run start.bat. On Linux/macOS, run ./start.sh. It sets everything up the first time — it even installs Node.js and Java for you if you don't have them. Your browser opens at http://localhost:8080.

Windows may say "Windows protected your PC" the first time — that's normal for free, unsigned apps and completely safe because the code is public on GitHub. Just click More info, then Run anyway. Tip: to skip it entirely, right-click the downloaded ZIP → Properties → click Unblock before extracting.
3

Set up & play

Choose a password, click Download to get a server (Paper or Vanilla), accept the EULA, and press Start. Open the Play Online tab to get a link your friends can join — no router setup needed.

The launcher installs Node.js and Java for you automatically if they're missing — nothing else to set up.

Frequently asked questions

Everything you might be wondering before you start.

Is it really free?

Yes — completely. MC Dashboard is open source and free to use forever. There are no subscriptions, no paywalls, and no ads. You only need a computer to run it on.

Do I need to know how to code?

No. Download the ZIP, unzip it, and double-click start.bat (Windows) or start.sh (Linux/macOS) — that's the whole installation. Everything after that is point-and-click in your browser.

Windows says it's "not safe" — is it?

Yes, it's safe. That warning is Windows SmartScreen, which flags any program that isn't code-signed — and code signing costs hundreds of dollars a year, so nearly all free open-source tools trigger it. The full source is public on GitHub, so you can verify exactly what it does. To allow it, right-click the downloaded .zip → Properties → click Unblock before extracting, or if you see "Windows protected your PC", click More info → Run anyway.

What do I need to run it?

Just a regular computer (Windows, Linux, or macOS) — even an old laptop or a Raspberry Pi works for a small server. The launcher installs everything it needs for you — including Java for the Minecraft server — so there's nothing to set up by hand.

How do my friends connect?

Easiest way: open the Play Online tab and click once — it sets up a free playit.gg link your friends paste into Minecraft, with no router or port-forwarding changes. You can also use your own IP on a home network.

Can I use plugins and mods?

Yes. Use a Paper server JAR for plugins or Fabric/Forge for mods, then search and install them straight from Modrinth inside the dashboard, or upload your own JARs.

Which Minecraft versions are supported?

Any recent Paper or Vanilla version — you can download them with one click from the dashboard, including the latest releases.

Is it safe to put online?

The dashboard is password-protected. For access beyond your home network, put it behind HTTPS or a VPN like Tailscale rather than exposing it directly. The README covers this in more detail.

Can I run more than one server?

Yes — create multiple server profiles, such as survival, creative, or modded, and switch between them. Each profile has its own world and settings.
